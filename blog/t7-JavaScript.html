Phase 0 Week 7 Technical Blog: Ruby Hashes vs. JavaScript Objects By: Nell Taylor | 11-22-14

Ruby hashes and JavaScript objects both contain sets of key-value pairs and both can serve the function of associative arrays, in which things in an array have a value associated with them. In JavaScript, objects go a step further in their usefulness than hashes; besides possessing an associative structure, the key-value pairs in an object serve as the definitive characteristics of that object. You can use the properties in objects to reference those characteristics as well as modify them, in which case the new value is stored in the property being modified. To reference the value of an object's property in JavaScript, you simply use object.property which will then access the value stored at whatever the key property is. You can also access a value using object["property"].
