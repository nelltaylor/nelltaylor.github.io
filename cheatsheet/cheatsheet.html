Cheatsheet: Looping in Ruby vs. JavaScript

Joseph Won & Nell Taylor

Ruby Examples

Ruby and JavaScript don't have 1:1 comparisons of every different type of loop. We'll show you Ruby loops first, then JavaScript.

Ruby: For Loops

Input

for i in 0..5
  puts i
end

Output

0
1
2
3
4
5

Ruby: While Loop

Input

count = 0

while count < 5 do
    puts "continuing program.."
    count += 1
end

Output

continuing program..
continuing program..
continuing program..
continuing program..
continuing program..

Ruby: Until Loop with Begin/End

Input

stars = 0

begin

  puts "counting the stars.."
  stars+=1

end until stars == 5

Output

counting the stars..
counting the stars..
counting the stars..
counting the stars..
counting the stars..

Ruby: Using the .each method

Input

(0..5).each do |i|
  puts i
end

Output

1
2
3
4
5

JavaScript Examples

These are the JavaScript examples.

JavaScript: For Loop -- multiple first statements

Input

for ( i = 0, word="hello", len =word.length; i<len; i++ ) {
  console.log(word.charAt(i))
  }

Output

h
e
l
l
o

JavaScript: For Loop -- Optional first statements

Input

i = 0;

for (; i<6; i++) {
    console.log(i);
}

Output

0
1
2
3
4
5

JavaScript: For/In Loop

Input

var person = {fname:"John", lname:"Doe", age:25};

var text = "";
var x;
for (x in person) {
    text += " "+person[x];
}

Output

' John Doe 25'

JavaScript: While Loop

Input

text = "";
i=0;

while (i < 6) {
    text += "bump ";
    i++;
}

console.log(text);

Output

bump bump bump bump bump bump

JavaScript: Other Ways to Use For

Input

friends = ["Jim","Henry","Sara","Hank"];
var i = 0;
var text = "";

for (;friends[i];) {
    text += friends[i] + "<br>";
    i++;
}

console.log(text)

Output

Jim
Henry
Sara
Hank
